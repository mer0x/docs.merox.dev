    Deploying Media Stack on Kubernetes with Longhorn and NFS Storage

    Prerequisites

    • A Kubernetes cluster
    • kubectl configured for your cluster
    • Longhorn installed on your Kubernetes cluster
    • An NFS server configured and accessible
    • Traefik Ingress controller installed in your Kubernetes cluster

  capacity:

  accessModes:

  nfs:

    apiVersion: apps/v1
kind: Deployment
metadata:
  name: jellyfin
  namespace: media
spec:
  replicas: 1
  selector:
    matchLabels:
      app: jellyfin
  template:
    metadata:
      labels:
        app: jellyfin
    spec:
      containers:
      - name: jellyfin
        image: jellyfin/jellyfin
        volumeMounts:
        - name: config
          mountPath: /config
        - name: videos
          mountPath: /data/videos
        ports:
        - containerPort: 8096
      volumes:
      - name: config
        persistentVolumeClaim:
          claimName: jellyfin-config
      - name: videos
        persistentVolumeClaim:
          claimName: jellyfin-videos
